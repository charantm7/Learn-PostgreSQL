# PostgreSQL Learning

This repository documents my PostgreSQL learning journey. It contains SQL queries, practice exercises, and notes covering PostgreSQL fundamentals to advanced concepts.

## 📚 Topics Covered

- Database & Table Creation
- Data Types
- Constraints
- CRUD Operations
- Filtering (`WHERE`, `LIKE`, `ILIKE`, `IN`, `BETWEEN`)
- Sorting (`ORDER BY`)
- Aggregate Functions
- `GROUP BY` & `HAVING`
- `DISTINCT` & `DISTINCT ON`
- Joins
- Transactions
- Views
- Indexes
- Functions
- Common Table Expressions (CTEs)
- Window Functions
- JSON & JSONB
- Database Design

## 🛠️ Environment

- PostgreSQL
- psql CLI
- Ubuntu Linux

- Practice production-ready database design and queries.
